# Week 1: Data Cleaning Assignment

## 📌 Objective
The objective of this assignment is to clean and preprocess a raw dataset to make it suitable for analysis.

---

## 📂 Files

- combined_dataset.csv → Raw dataset  
- cleaned_dataset.csv → Final cleaned dataset  
- Data_Cleaning_Assignment.ipynb → Jupyter Notebook containing the cleaning process  

---

## 🧹 Data Cleaning Steps

- Removed duplicate records  
- Handled missing values (removed or filled where necessary)  
- Converted data types (e.g., price columns to numeric format)  
- Cleaned columns such as price and ratings by removing unwanted symbols  
- Created a new column: `total_amount`
